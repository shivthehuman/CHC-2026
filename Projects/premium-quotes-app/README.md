# 🌟 ZenQuotes - Premium Quote Gallery

A modern, responsive, and elegant quote discovery application built for the Web Dev Cohort 2026. This application interfaces seamlessly with the FreeAPI to deliver inspiring quotes wrapped in a beautiful UI.

## ✨ Key Features
- **Dynamic Hero Generator:** Instantly fetches and displays a random "Quote of the Day" with smooth fade-in animations.
- **Masonry-Style Grid:** A beautifully structured gallery view of quotes using modern CSS Grid and Tailwind breakpoints.
- **Native Dark Mode:** Fully integrated Dark/Light mode toggle that respects user preference and enhances readability.
- **Graceful Error Handling:** Enterprise-level loading skeletons (`animate-pulse`) and elegant error fallback states for API downtime.
- **Responsive Design:** Mobile-first approach ensuring the UI looks pristine on phones, tablets, and massive desktop monitors.

## 🛠️ Tech Stack
- **Structure:** Semantic HTML5
- **Styling:** Tailwind CSS (via CDN) + Custom CSS (`style.css`) for scrollbars and keyframe animations.
- **Logic:** Vanilla JavaScript (ES6+ `async/await`, Fetch API, DOM Manipulation)
- **Data Source:** [FreeAPI Quotes Endpoint](https://api.freeapi.app/api/v1/public/quotes)

## ⚙️ Local Setup
1. Clone the repository: `git clone <your-repo-link>`
2. Open the folder in your terminal: `cd premium-quotes-app`
3. Launch via Live Server (VS Code Extension) or simply open `index.html` in your browser. No `npm install` required!
