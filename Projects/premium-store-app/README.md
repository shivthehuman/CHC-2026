# 🛒 @shivthehuman - Premium Store Interface

A high-performance product listing interface built for the Web Dev Cohort 2026. This application seamlessly integrates with the FreeAPI Random Products endpoint to deliver a modern, Amazon-style e-commerce browsing experience.

## ✨ Key Features
- **Modern E-Commerce Layout:** Fully responsive grid architecture optimized for desktop, tablet, and mobile.
- **Micro-Interactions:** Subtle hover zoom effects on product images and active scale states on buttons.
- **Enterprise Error Handling:** Elegant skeleton loaders (`animate-pulse`) and image fallback configurations (`onerror`).
- **Native Dark Mode:** Deeply integrated theme toggling using Tailwind CSS class strategy.
- **Personal Branding:** Custom integrated footer and branding elements.
