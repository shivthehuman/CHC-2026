# Developer Tool Landing Page - Cursor Clone

## Overview
This is a desktop-first HTML/CSS recreation of the Cursor.com landing page, created for the Web Dev Cohort 2026 assignment.

## Sections Recreated
All 11 required sections have been fully built exactly as requested:
1. Top Navigation Bar
2. Hero Section
3. Trusted By / Logos
4. Feature Sections (3 alternating blocks)
5. Feature Cards Section
6. Testimonials
7. Use Cases / Stories
8. Changelog / Updates
9. Team / About
10. Final CTA
11. Footer

## Technical Details
- **Tech Stack:** HTML5 & CSS3 ONLY.
- **Constraints Followed:** No JavaScript, No TailwindCSS, Desktop-only layout (`min-width: 1024px` enforced), No animations.
- **Fonts:** `Inter` font imported via Google Fonts to strictly match Cursor's visual hierarchy. 
- **Colors:** - Background: `#000000` & `#0a0a0a`
  - Text Primary: `#ffffff`
  - Text Secondary: `#a1a1aa`
  - Borders/Accents: `#27272a`
- **Layout Approach:** Heavy use of modern CSS Flexbox for alternating rows/navigation and CSS Grid for uniform card layouts. Semantic HTML tags (`<nav>`, `<header>`, `<section>`, `<footer>`) used throughout.
